# Homebrew Tap for Vasa CLI

Official Homebrew tap for the [Vasa](https://tryvasa.com) command-line interface.

## Install

```bash
brew tap nordlake/tap
brew install vasa
```

## Upgrade

```bash
brew upgrade vasa
```

## Alternative: Install Script

```bash
curl -sSL https://get.tryvasa.com | sh
```

The install script detects your OS and architecture, downloads the latest release, verifies the checksum, and installs the `vasa` binary.

## Verify

```bash
vasa version
```
